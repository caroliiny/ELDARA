# Descrição
Eldara  é um jogo desenvolvido na Godot Engine como projeto da disciplina de Computação Gráfica.  
O objetivo do trabalho foi aplicar, de forma prática, conceitos estudados em sala, integrando elementos gráficos, lógica de programação e interação em tempo real dentro de um ambiente de jogo.

O projeto explora a construção de cenas, uso de assets gráficos, organização estrutural e renderização, buscando consolidar o entendimento dos fundamentos da computação gráfica no contexto do desenvolvimento de jogos.

# Tecnologias Utilizadas
- Godot Engine
- GDScript
- Assets gráficos gratuitos de bibliotecas online

# Conceitos Trabalhados
Durante o desenvolvimento do jogo, foram aplicados conceitos fundamentais da computação gráfica, incluindo:

- Estruturação de cenas e hierarquia de nós
- Renderização de elementos gráficos em tempo real
- Uso de sprites, texturas e materiais
- Transformações geométricas (posição, escala e rotação)
- Controle de câmera e visualização
- Interação do usuário com o ambiente gráfico
- Organização e otimização básica do projeto

Esses conceitos foram integrados diretamente à construção do cenário e dos elementos interativos do jogo.

## Estrutura do Projeto
O projeto foi organizado de forma a facilitar a leitura, manutenção e entendimento do código.
